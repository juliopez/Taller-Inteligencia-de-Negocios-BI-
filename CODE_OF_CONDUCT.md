# Code of Conduct

## Purpose

This repository contains academic material for the course **Business Intelligence**.

The goal of this repository is to provide an open and respectful environment for
sharing educational resources related to data analysis, business intelligence,
and decision support systems.

---

## Expected Behavior

Participants using this repository are expected to:

- Use the material in an ethical and responsible way.
- Respect the intellectual work of others.
- Maintain respectful communication when interacting through issues or discussions.
- Use the repository for educational purposes.

---

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or discrimination of any kind.
- Offensive or inappropriate language.
- Misuse of the repository for purposes unrelated to education.

---

## Academic Integrity

Students using this repository must respect academic integrity policies of their institutions.

The materials provided here are intended to support learning and understanding of
Business Intelligence concepts.
